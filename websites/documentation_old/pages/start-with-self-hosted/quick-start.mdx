# Quick start

The quick start aims to people that want to get running fast, with as few steps as possible.

<Callout type="info" emoji="ℹ️">
  This guide assumes you have [Docker](https://www.docker.com/) and [Docker
  Compose](https://docs.docker.com/compose/) installed
</Callout>

## Step by step

<Steps>

### Get the project running

In your favorite terminal, run the following commands:

```shell
$ git clone https://github.com/progressively-crew/progressively
$ cd progressively
$ docker-compose up -d
```

### Create your admin user

You can now open [http://localhost:3000/welcome](http://localhost:3000/welcome) to setup your <strong>admin</strong> user.

</Steps>
