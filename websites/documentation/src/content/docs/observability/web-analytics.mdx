## Introduction to Web Analytics

Web analytics is the measurement, collection, analysis, and reporting of web data to understand and optimize web usage. It helps you gain insights into user behavior, track key performance indicators (KPIs), and make data-driven decisions.

## Key Metrics

1. **Page Views**: Number of times a page is viewed.
2. **Unique Visitors**: Number of distinct users visiting your site.
3. **Bounce Rate**: Percentage of visitors who leave the site after viewing only one page.
3. **Custom events**: Number of pre-defined (by you) events triggered during the navigation.
